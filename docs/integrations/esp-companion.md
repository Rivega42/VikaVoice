# Мини-экран-компаньон (ESP32)

> Статус: 🔲 backlog (EPIC-8, после продуктовой версии) · Обновлено: 2026-07-07

## Идея

Помимо основного 7" экрана в продуктовой версии — **дешёвый компаньон** на базе
GeekMagic SmallTV Pro (ESP32) или аналога.

**Зачем:**

- Стоит на столе рядом, показывает состояние Вики
- Индикация: «слушает» / «думает» / «говорит» / **красный «приватный режим»**
- Если сама Вика в другой комнате — компаньон рядом, чтобы видеть статус
- Стоит 300–700 ₽ вместо 4–6 тыс. ₽ за большой экран

**Что показывает:** иконку состояния, текущее время, (опц.) короткий текст последней
реплики, (опц.) индикатор аудио-активности.

## Технический подход

- ESP32 подключается по Wi-Fi к устройству
- Опрашивает `GET /status` каждые ~500 мс или подписывается на WebSocket
  (эндпоинт статуса — часть API v1, EPIC-7)
- Своя прошивка на Arduino-стеке

## Кандидаты железа (~300–700 ₽)

- **GeekMagic SmallTV Pro** — ESP32-WROOM-32, IPS-экран, Wi-Fi (AliExpress)
- **NEIMZ мини (NZ1237)** — клон SmallTV Ultra, ~330 ₽
- **GeekMagic GIFTV Crystal Cube** — та же начинка в стеклянном кубике: эффектнее, но тусклее
