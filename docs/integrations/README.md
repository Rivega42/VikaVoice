## Календарь и задачи

Вика знает расписание, напоминает, планирует, добавляет задачи по голосу.
Кандидаты: Google Calendar, Яндекс.Календарь, Notion, Todoist, Trello (для команд).
Уже частично работает через OCPlatform-инструменты — в VikaVoice просто пробрасываем.
**Отдельной работы не требуется.**

## Мессенджеры

Отправка протоколов встреч, задач, напоминаний. Приоритет: Telegram (работает через
OCPlatform), затем Notion (команды), email (юристы — консервативная аудитория).

## Не берём

- **Прошивки готовых ESP-проектов целиком** — они под ESP, у нас RPi/Linux. Полезны идеи
  (OAuth-токен УДЯ, AP-режим), сам код — нет.
- **Alice SDK** — не хотим зависеть от Яндекса как платформы; УДЯ используем только как
  API умного дома.
- **Home Assistant** — потенциальный «мост» к большой экосистеме устройств, но не
  приоритет (backlog).
