## Компоненты

### Локально на RPi4

| Компонент | Технология | Зачем локально |
|-----------|-----------|----------------|
| Wake-word | openWakeWord / Porcupine | Всегда слушает, без интернета |
| STT | whisper.cpp (tiny или base) | Скорость + приватность |
| TTS | Piper (ru голос) | Скорость, нет задержки API |
| VAD | silero-vad | Детект конца фразы |
| Diarization | pyannote (для режима встречи) | Кто говорит |

### В облаке (OCPlatform)

| Компонент | Зачем в облаке |
|-----------|----------------|
| LLM (Claude/DeepSeek) | Вычислительно тяжело для RPi |
| Память Вики | Сохраняется между сессиями |
| Инструменты (календарь, задачи) | Интеграции |
| Личность и контекст | SOUL.md, MEMORY.md |

## Навыки Вики

Детальное описание навыков — см. [SKILLS.md](SKILLS.md).

Краткая справка:

| Навык | Триггер | Активная роль | Данные наружу |
|-------|---------|---------------|---------------|
| 🏠 Home | wake-word «Вика» | разговор | → OCPlatform |
| 🎯 Facilitator | «начнём встречу» | модерация встречи | → OCPlatform |
| 📝 Transcriber | «просто запиши» | молча пишет + суммирует | → OCPlatform |
| 🔒 Private | «приватный режим» | модификатор поверх остальных | **никуда** |

### Приватный режим — техническая реализация

Локальная LLM запускается на самом RPi4 через `llama.cpp`:

| Модель | Размер (GGUF Q4) | RAM | Скорость на RPi4 8GB |
|--------|------------------|-----|----------------------|
| llama-3.2-1B | ~700 MB | ~1.5 GB | ~10 tok/s |
| llama-3.2-3B | ~2 GB | ~3.5 GB | ~4 tok/s |
| phi-3.5-mini (3.8B) | ~2.3 GB | ~4 GB | ~3 tok/s |

При активации:
1. Роутер запросов переключается: `OpenClawClient` → `LocalLlamaClient`
2. LED-кольцо → красное
3. (опционально в продуктовой версии) физическая airgap-кнопка отрубает сетевой стек
4. Whisper остаётся тем же (он уже локальный)
5. Piper остаётся тем же (уже локальный)

Выход:
- Голосом: «Вика, обычный режим»
- Таймер, заданный при активации
- Физическая кнопка

## Голосовой стек (детали)

### STT варианты
| Модель | RAM | Скорость | Качество RU |
|--------|-----|----------|-------------|
| whisper.cpp tiny | ~200MB | очень быстро | удовлетворительно |
| whisper.cpp base | ~300MB | быстро | хорошо |
| whisper.cpp medium | ~1.5GB | медленнее | отлично |

На RPi4 8GB рекомендуется **base** для баланса скорости/качества.

### TTS варианты
| Движок | Латентность | Качество RU | Лицензия |
|--------|-------------|-------------|---------|
| Piper (irina) | ~50ms CPU | хорошо | MIT |
| Piper (ruslan) | ~50ms CPU | хорошо | MIT |
| CosyVoice 2 | ~150ms GPU | отлично + клонирование | Apache-2.0 |
| OpenAI TTS | ~300ms API | отлично | SaaS |

Для MVP: **Piper** (CPU, MIT, быстро). Потом можно добавить клонирование голоса через CosyVoice 2.

### Wake-word
| Движок | Ложные срабатывания | CPU |
|--------|---------------------|-----|
| openWakeWord | среднее | низкое |
| Porcupine | низкое | очень низкое |
| Sherpa-ONNX | низкое | низкое |

Рекомендуется **openWakeWord** (open-source) или **Sherpa-ONNX** с кастомной моделью "Вика".

## Интеграция с OCPlatform

OCPlatform gateway уже поднят: `http://185.23.239.126:<port>/v1/chat/completions`

RPi4 отправляет стандартный OpenAI-совместимый запрос.
OCPlatform подхватывает контекст, память, инструменты — возвращает ответ.

Это означает: **Вика в домашнем устройстве = та же Вика, что в Telegram**.
Память общая. Личность одна.
