## Milestone 1 — MVP (открытая сборка) ← следующий

**Цель:** работающее устройство на столе, провода торчат, но разговаривает.

- [ ] Закупить железо по BOM (~24к ₽)
- [ ] Установить Raspberry Pi OS
- [ ] Установить whisper.cpp (base модель, русский)
- [ ] Установить Piper TTS (голос ru_irina или ruslan)
- [ ] Настроить openWakeWord с моделью "Вика" (или Sherpa-ONNX)
- [ ] Написать клиент: wake-word → STT → OCPlatform → TTS → колонка
- [ ] Протестировать latency (цель: ответ < 3 сек end-to-end)
- [ ] Режим фасилитатора: добавить diarization (pyannote)
- [ ] Веб-интерфейс реалтайм транскрипта (FastAPI + WebSocket)

**Критерий готовности:** разговор голосом с Викой без телефона работает.

## Milestone 2 — Режим транскрайбера

**Цель:** ставишь на стол → нажимаешь "запись" → получаешь протокол.

- [ ] Пассивный режим (нет wake-word, нет TTS-ответов)
- [ ] Diarization: "Спикер 1", "Спикер 2", ...
- [ ] В конце: LLM суммаризация → структурированный протокол
- [ ] Экспорт: Markdown, отправка в Telegram

## Milestone 3 — Продуктовая версия

**Цель:** красиво выглядит, можно показать инвестору.

- [ ] Брендинг корпуса
