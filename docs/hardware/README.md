## Что НЕ входит в BOM (отдельные статьи затрат)

- Разработка ПО: распознавание речи, диаризация, запоминание голоса, смысловой анализ.
- Серверное оборудование для облачной и on-premise редакций.
- Кастомный микрофонный массив 6–8 микрофонов на XVF-3800 (разработка платы и прошивки).
- Пресс-форма корпуса для серийного производства.
- Сертификация (декларирование соответствия, ЕАС); меры по защите биометрических данных
  ([152-ФЗ](../compliance/152fz.md)).
- Логистика, таможенные пошлины, НДС.
- Лицензии и модели при использовании платных сервисов ASR / языковых моделей.

## Допущения

- Валюта — доллары США; цены ориентировочные, для прототипной партии (1–10 шт),
  без доставки, пошлин и НДС.
- Начало 2026: дефицит DRAM/NAND — вычислитель и накопители временно дороже;
  при нормализации рынка себестоимость снизится.
- Микрофонный массив: прототип — готовый USB-модуль с аппаратным DSP; серия —
  собственная плата 6–8 MEMS (XVF-3800): дешевле в объёме, но требует NRE
  ([ADR-0002](../adr/0002-mic-array.md)).
- При серийных объёмах комплектующие дешевле ориентировочно на 25–45%.

## Правила ведения

- Новая конфигурация = новый CSV-лист + строка в сводной таблице выше.
- Цена меняется — правим CSV и сводку в одном PR; в других документах цифры
  не повторяются, только ссылки сюда.
- Решения о железе (платформа, микрофоны и т.п.) — через [ADR](../adr/README.md),
  BOM только отражает принятое.
