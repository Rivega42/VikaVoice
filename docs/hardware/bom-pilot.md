# BOM: пилотная партия — 3 устройства, корпус SLA (фотополимерная смола)

> Статус: ✅ актуален · Обновлено: 2026-07-07 · Связанные ADR:
> [0001](../adr/0001-compute-platform.md), [0002](../adr/0002-mic-array.md) ·
> Детализация: [bom-pilot-3x.csv](bom-pilot-3x.csv)

Закупочный список для сборки **минимум 3 устройств** Edge-конфигурации
(EPIC-4 → EPIC-8, пилоты). Корпуса печатаются **жидким фотополимером (SLA/MSLA)**
из бутылок смолы 1 кг — это прототипный вариант до пресс-формы
(серия — литьё под давлением, NRE в [bom-notes.md](bom-notes.md)).
Геометрия корпуса и решётка — [hardware/INDUSTRIAL-DESIGN.md](../../hardware/INDUSTRIAL-DESIGN.md)
(цилиндр Ø150×H180 — расход смолы см. ниже, оценка дана для этой геометрии
с запасом); кожух и ткань — [hardware/CASE-COVER.md](../../hardware/CASE-COVER.md).

## Расход смолы

Оценка на один корпус (цилиндр ~Ø110×160 мм, стенка 2,5 мм, две крышки,
внутренние стойки, диффузор LED-кольца):

| Статья | Масса |
|--------|------:|
| Детали корпуса | ~260–300 г |
| Поддержки и подложка | +20–25% |
| Брак / повторная печать | резерв ~15% |
| **Итого на устройство** | **~400–450 г** |
| **На 3 устройства** | **~1,3 кг → 2 бутылки по 1 кг** |

## Замечания по материалу

- Брать **инженерную смолу (ABS-like / Tough)**, не стандартную: стандартная
  хрупкая — резьбовые стойки и защёлки ломаются при сборке.
- Цвет — тёмный непрозрачный корпус + отдельно полупрозрачная деталь диффузора
  (можно допечатать из остатка стандартной прозрачной смолы, ~30 г).
- Акустические окна микрофонов печатать с запасом по диаметру и закрывать
  тканью (поз. 13 bom-edge) — смола не даёт «акустической сетки».
- УФ-дотверждение обязательно (станция или лампа 405 нм), иначе корпус
  желтеет и ведёт геометрию.
- Если поле принтера меньше высоты корпуса — печатать цилиндр из двух секций
  со стыковкой на стойках (учтено в резерве расхода).

## Что НЕ входит

Те же исключения, что в [bom-notes.md](bom-notes.md): ПО, серверы, сертификация,
логистика/пошлины/НДС, а также электроэнергия и труд печати/постобработки.
