# BOM: домашний MVP (dev-конфигурация на RPi4)

> Статус: 🟡 историческая конфигурация — **устарела относительно [ADR-0001](../adr/0001-compute-platform.md)**
> (продуктовая база Edge — RPi5, см. [bom-edge.csv](bom-edge.csv)); сохранена как дешёвый
> dev-вариант из уже имеющегося железа · Обновлено: 2026-07-07 · Связанные ADR:
> [0001](../adr/0001-compute-platform.md), [0002](../adr/0002-mic-array.md)

> Железо для личного MVP (открытая сборка на столе, без корпуса). Одна платформа
> покрывает оба режима: домашняя Вика + фасилитатор.

## Решения по ключевым выборам

Формат устройства — «спикерфон»: цилиндр в центре стола, круговой микрофон 360°, экран
сверху в продуктовой версии (как Meeting Owl / Jabra Speak — то, что люди покупают).
Выбор микрофонного массива и отклонённые альтернативы (AC108, ESP32-версии, готовые
спикерфоны) — [ADR-0002](../adr/0002-mic-array.md).
