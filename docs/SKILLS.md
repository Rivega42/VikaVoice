# Навыки Вики

> Одна личность, несколько ролей. Переключение — голосом или контекстно.

## Общее

- Все навыки используют **одно и то же железо** (RPi4 + ReSpeaker XMOS + колонка)
- Все используют **одну и ту же Вику** (память, характер, инструменты в OCPlatform)
- Отличаются только: что она делает + куда идут данные

---

## 🏠 Навык 1 — Домашний ассистент

**Триггер:** wake-word «Вика» → сразу разговор.

**Пайплайн:**
```
wake-word → микрофон → whisper.cpp (RPi) → OCPlatform gateway → 
Piper TTS (RPi) → колонка
```

**Данные уходят:** транскрипт запроса → OCPlatform (шифрованно по TLS).

**Индикация:** синий пульс на LED-кольце во время разговора.

**Использование:**
- Утром: «Вика, что у меня сегодня?»
- В комнате: «Вика, напомни через час выпить воды»
- Вечером: «Вика, что нового в GrandHub?»

---

## 🎯 Навык 2 — Фасилитатор встреч

**Триггер:** голосовая команда «Вика, начнём встречу» или кнопка на устройстве.

**Пайплайн:**
```
непрерывная запись → VAD → diarization (pyannote) → 
whisper.cpp → OCPlatform → активные реплики от Вики → 
Piper TTS → колонка + реалтайм-транскрипт на экране
```

**Что делает Вика:**
- Следит за таймингом («мы 20 минут на этой теме, идём дальше?»)
- Задаёт уточняющие вопросы, если решение неоднозначно
- Выявляет action items и назначает ответственных
- Модерирует брейншторм (собирает идеи, потом группирует)
- В конце: протокол + список задач → отправка в TG/Notion/Trello

**Данные уходят:** полный транскрипт → OCPlatform.

**Индикация:** зелёное вращение LED (запись + анализ).

---

## 📝 Навык 3 — Транскрайбер

**Триггер:** «Вика, просто запиши» или кнопка "REC".

**Пайплайн:**
```
непрерывная запись → VAD → diarization → whisper.cpp → 
сохранение локально → по окончании: LLM суммаризация → протокол
```

**Отличие от фасилитатора:**
- Вика молчит (нет TTS-ответов в реальном времени)
- Не задаёт вопросов, не модерирует
- Только запись + структурирование в конце

**Данные уходят:** транскрипт → OpenClaw для суммаризации (или локально в приватном режиме).

**Индикация:** белое ровное свечение (только запись).

---

## 🔒 Навык 4 — Приватный режим

**Триггер:** «Вика, приватный режим» — работает поверх любого из навыков 1-3.

**Пайплайн (полностью локальный):**
```
микрофон → whisper.cpp (base RU) → 
локальная LLM на RPi (llama.cpp, tiny/small модель) → 
Piper TTS → колонка
```

**Ключевое ограничение:** **ничего не уходит в сеть.** Ethernet/Wi-Fi физически можно отрубить (планируется физическая кнопка "airgap" в продуктовой версии).

**Что теряется:**
- Умная память (долгосрочная в OCPlatform) — недоступна
- Инструменты (календарь, отправка в Notion) — недоступны
- Качество ответов ниже (маленькая LLM ~1-3B параметров)

**Что остаётся:**
- Транскрипция (whisper.cpp base = хорошее качество)
- Базовые ответы (llama-3.2-3B или phi-3.5-mini в GGUF)
- Суммаризация встречи (простая, без внешних инструментов)

**Индикация:** **красное** свечение LED-кольца — визуально видно, что режим включён.

**Выход из режима:** «Вика, обычный режим» или таймер (задаётся при активации: "приватный режим на 2 часа").

**Целевое использование:**
- Психолог с клиентом
- Юрист с клиентом
- Врач на приёме
- Домашние разговоры, которые не хочется отдавать в облако
- Совещания под NDA

---

## Переключение между навыками

| Как | Пример |
|-----|--------|
| Голосом | «Вика, начнём встречу», «Вика, приватный режим» |
| Контекстно | Утром сама включает «home» режим, при обнаружении >2 голосов предлагает «facilitator» |
| Кнопкой | Physical button на устройстве (в продуктовой версии) |

Приватный режим — **модификатор**, накладывается на любой из первых трёх.
Пример: «Вика, начнём встречу в приватном режиме» → фасилитатор + всё локально.
