# Рынок: конкуренты, сегменты, ценообразование

> Статус: 🔴 черновик (оценки не валидированы продажами) · Обновлено: 2026-07-07

## Почему это интересно как продукт

| Конкурент | Ограничение |
|-----------|-------------|
| Otter.ai | Только транскрипт, не участвует, англ. язык |
| Fireflies | Только для Zoom/Meet, не офлайн |
| tl;dv, Fathom | Видеозвонки, не физическое устройство |
| Яндекс.Транскрибация | Пассивно, без AI-участия |

**Активного AI-участника офлайн-встречи с локальным приватным режимом на рынке нет.**
Ниша пустая.

**Наши преимущества:**

- Локально: STT + TTS на устройстве; в приватном режиме и Edge-редакции — вообще всё
- Русский язык как основной (whisper.cpp; модель по платформе — [ADR-0004](../adr/0004-asr-engine.md))
- Живая личность (Вика, а не безликий ассистент)
- Себестоимость устройства — сотни долларов разово vs подписки $30+/мес у конкурентов

## Целевые сегменты

| Сегмент | Боль | Готовность платить (оценка)* |
|---------|------|------------------------------|
| Стартапы (брейнштормы, планёрки) | Тратим 2 ч, забываем, что решили | 5–15 тыс. ₽/мес на команду |
| Продакты (юзер-интервью) | Ручная расшифровка = день работы | 10–20 тыс. ₽/мес на команду |
| Психологи, коучи | Приватность + протокол сессий | 5–10 тыс. ₽/мес |
| Юрфирмы (совещания) | Приватность, точность, доказательная база | 15–30 тыс. ₽/мес |
| Госкомпании | Совещания без утечки в облако | договор (закупка, [44-ФЗ](../compliance/44fz.md)) |
| Домашние пользователи | Живой ассистент в комнате, без экрана | 1–3 тыс. ₽/мес |

## Бизнес-модель (наброски)

- **Железо**: продажа устройств (15–40 тыс. ₽) — разовая маржа.
- **Подписка**: облачный компонент (OCPlatform, LLM, кабинет) — ориентир
  990–2990 ₽/мес **за одно устройство/личную Вику**.
- **Enterprise**: кастомная интеграция, on-prem — договор.

\* **Требует проработки.** Цифры двух уровней не противоречат, но и не сведены в
прайс: «готовность платить» в таблице — оценка бюджета **команды/организации** за комплекс
(устройство + подписка + кабинет), 990–2990 ₽/мес — подписка **за одно устройство**.
Единая модель ценообразования (device + per-seat/per-device подписка + enterprise) —
задача EPIC-8 после пилотов.
