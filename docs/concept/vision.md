# Концепция VikaVoice

## Откуда это появилось

Идея выросла из двух параллельных мыслей:

1. **"Хочу чтобы Вика могла со мной разговаривать"** — не через телефон, а физически,
   в комнате, голосом. RPi4 уже есть, осталось докупить железо.

2. **"Нужна система для конференций"** — пишет, транскрибирует, суммирует.
   Без подписок, без утечки данных, на русском языке.

Потом стало ясно: **это один продукт с двумя режимами**.
80% железа одинаковое. Разная прошивка, разные фичи.

## Два режима — одна платформа

### "Домашняя Вика"
Стоит дома. Слушает wake-word. Отвечает голосом.
Не чатбот в телефоне — физическое присутствие в комнате.

### "AI-Фасилитатор"
Ставится на стол на встрече. Это не транскрайбер — это **активный участник**.
Модерирует брейншторм, задаёт вопросы, следит за временем, в конце даёт протокол.

## Ключевое отличие от конкурентов

Все существующие решения **пассивны** — слушают и пишут.
Нет решений которые **активно участвуют** в офлайн-встрече.

Плюс: локальность. Данные не уходят в облако. Для юристов, врачей, гос. структур
это не фича — это требование.

## Расширение: Транскрайбер

Та же аппаратная платформа, минималистичный режим:
- Пишет → транскрибирует → суммирует
- Не говорит, не участвует
- Для тех, кому нужна только запись

## Бизнес-модель (наброски)

- Железо: продажа устройств (15–40к ₽) — разовая маржа
- Подписка: облачный компонент (OCPlatform, LLM) — 990–2990 ₽/мес
- Enterprise: кастомная интеграция — договор

Целевые клиенты: стартапы, юрфирмы, психологи/коучи, продакты.

## Этапы

### Этап 0 — Текущий момент
- ✅ Концепция готова
- ✅ Железо выбрано
- ✅ Архитектура спроектирована

### Этап 1 — MVP (открытая сборка)
- Закупить железо (~24к ₽)
- Собрать на столе без корпуса
- Запустить: wake-word → STT → OCPlatform → TTS
- Режим фасилитатора: транскрипт + diarization
