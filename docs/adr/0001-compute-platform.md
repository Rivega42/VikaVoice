# 0001. Вычислительная платформа устройства

- Статус: **accepted**
- Дата: 2026-07-07

## Контекст и проблема

В репозитории сосуществовали два противоречащих решения: ранние документы (архитектура,
домашний BOM) строились на **Raspberry Pi 4 8GB** (~24 000 ₽ за MVP), а инженерный пакет
переговорного комплекса (BOM Edge) — на **Raspberry Pi 5 8GB** (+ опция нейроускорителя
Hailo-8L) и **Pi Zero 2 W** для тонкого устройства. Нужна одна платформенная линейка.

Требования: локальный ASR (whisper.cpp) в разумном realtime, диаризация и enrollment на
борту (Edge), локальная LLM 1–3B для приватного режима, USB для микрофонного массива,
доступность в РФ.

## Рассмотренные варианты

1. **Raspberry Pi 5, 8 ГБ** — заметно быстрее RPi4 (~2–3× CPU), PCIe → NVMe и
   нейроускоритель M.2 (Hailo-8L), актуальная платформа. Минус: дороже (дефицит DRAM 2026),
   выше энергопотребление/нагрев.
2. **Raspberry Pi 4, 8 ГБ** — уже есть в наличии, дешевле; whisper.cpp `base` работает,
   но медленнее; нет PCIe. Достаточно для домашнего MVP и разработки.
3. **Pi Zero 2 W** — только захват и стриминг (тонкое устройство), 512 МБ RAM — обработка
   на борту исключена.
4. **NVIDIA Jetson (Orin Nano)** — тяжёлый ИИ на борту (Parakeet, крупные модели), но
   дороже, сложнее питание/теплоотвод, доступность в РФ хуже.
5. **Мини-ПК x86 (N100)** — быстрее RPi5 за сравнимые деньги, но хуже форм-фактор,
   GPIO/энергопотребление, «не embedded».

## Решение

Линейка из трёх ролей:

- **Edge-тир (толстое устройство): Raspberry Pi 5, 8 ГБ** — базовая плата продукта;
  опционально Hailo-8L. BOM: [`docs/hardware/bom-edge.csv`](../hardware/bom-edge.csv).
- **Dev/домашний MVP: Raspberry Pi 4, 8 ГБ** — допустимая плата для разработки и личного
  устройства из имеющегося железа; продуктовые замеры производительности делаются на RPi5.
  BOM: [`docs/hardware/bom-mvp.md`](../hardware/bom-mvp.md).
- **Тонкий тир: Raspberry Pi Zero 2 W** (при 6–8 аудиоканалах — допустим RPi4 2 ГБ) —
  только захват и потоковая передача. BOM: [`docs/hardware/bom-thin.csv`](../hardware/bom-thin.csv).

Jetson — отложенная опция внутри толстого тира, если понадобится тяжёлый ИИ на борту;
отдельным ADR при необходимости.

## Последствия

- Все документы приведены к этой линейке; упоминания «RPi4 как продуктовая платформа Edge»
  устарели.
- Код не должен зависеть от платы: конфигурация выбирает модели по доступной RAM/CPU.
- Бюджеты производительности (латентность ASR/LLM) фиксируются для RPi5; на RPi4 они
  информативные.
