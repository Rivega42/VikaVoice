# 0002. Микрофонный массив

- Статус: **accepted**
- Дата: 2026-07-07

## Контекст и проблема

Ключевое аппаратное преимущество продукта — круговой микрофонный массив 360° с аппаратной
обработкой: эхоподавление (AEC), формирование луча (beamforming), определение направления
на говорящего (DOA), шумоподавление. В документах фигурировали два разных готовых модуля
(ReSpeaker XVF3800 4-Mic и ReSpeaker USB Mic Array v2.0 на XVF-3000) — нужно зафиксировать
линию: прототип → серия.

## Рассмотренные варианты

1. **ReSpeaker USB Mic Array v2.0 (XMOS XVF-3000, 4 мик., USB)** — аппаратные AEC/DOA/
   beamforming на чипе, встроенное LED-кольцо (12 шт.), plug&play как USB-аудиоустройство,
   проверенная классика. Минус: снят с производства у части дистрибьюторов, 4 микрофона.
2. **ReSpeaker XMOS XVF3800 4-Mic (плата, USB)** — новее, тот же класс DSP-обработки,
   доступен на маркетплейсах. Минус: меньше готовых интеграций, дороже.
3. **ReSpeaker 4/6-Mic HAT (AC108)** — дёшево, но **без DSP**: AEC/beamforming пришлось бы
   делать программно на CPU; 6-мик HAT снят с поддержки (до Pi 4). Отклонено.
4. **ESP32-версии массивов / Lite Kit (ESPHome)** — автономный контроллер ломает
   архитектуру с RPi; экосистема Home Assistant — не наша. Отклонено.
5. **Готовые спикерфоны (Anker PowerConf S3 и т.п.)** — хорошие AEC/микрофоны, но нет DOA
   наружу, недоступны в РФ, «чужой продукт внутри продукта». Отклонено.

## Решение

- **Прототип:** готовый USB-модуль с аппаратным DSP семейства XMOS XVF —
  **XVF-3000 (ReSpeaker USB Mic Array v2.0)** или **XVF3800 4-Mic (плата)**;
  выбор конкретного модуля — по фактической доступности на момент закупки, интерфейс
  одинаковый (USB-аудио + DOA-метаданные). Расчётный BOM использует XVF-3000.
- **Серия:** собственная плата **6–8 MEMS-микрофонов на XMOS XVF-3800**
  (NRE ≈ $15–40 тыс. — см. [единый BOM](../hardware/README.md)); даёт ≥6 микрофонов
  (требование черновика ТЗ), контроль формы/акустики корпуса и снижение себестоимости в объёме.

## Последствия

- Слой `software/core/mic_array/` абстрагирует источник: «обработанный моно-луч + DOA-метки»,
  чтобы смена модуля/переход на свою плату не трогали остальной конвейер.
- LED-кольцо: у XVF-3000 есть встроенное (12 LED) — внешнее кольцо в BOM помечено опцией.
- Разработка своей платы (schematics, прошивка XVF-3800) живёт в `hardware/` — начнётся не
  раньше пилотов (EPIC-8).
