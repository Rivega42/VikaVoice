# Кожух (тканевая обшивка) — DIY-опыт сообщества

> Что можно взять из опыта людей, которые сами перешивали Яндекс.Станцию.
> Источник: [yaboard.com wiki](https://wiki.yaboard.com/w/3D-модель_кожуха_для_Яндекс.Станции)

## Ключевые находки для VikaVoice

### 1. Заводской кожух Яндекса КРИТИКУЮТ 🚨

По отзывам пользователей: **«оригинальный пластиковый кожух с плотной тканью
делает звук очень гулким, снижает громкость и чёткость ВЧ-динамиков... без стокового
кожуха нет "бубнежа" и появляются чёткие высокие частоты»**.

**Что это значит для нас:**
- Яндекс сам сделал компромисс: плотная ткань = защита, но акустика хуже
- **У нас задача проще** — не музыкальная колонка, а голосовой ассистент
- Значит для нас **не нужна максимальная акустическая прозрачность**, но
  и **плотная ткань может задушить микрофоны** (что критичнее чем звук из
  динамика)
- Микрофоны ReSpeaker чувствительны к плотности ткани над ними

**Вывод:** ткань выбираем по **микрофонной прозрачности**, а не только по вкусу.
Тест: положить ткань поверх ReSpeaker и замерить SNR при речи с 2-3м.

### 2. Механика съёма кожуха от Яндекса — заимствуем 💡

Как снимается кожух Станции:
> - Слегка сжать Станцию с боков ладонями → кожух сдвинется вниз
> - Перевернуть → снять до конца

**Это гениально:** нет винтов, нет защёлок для отвёртки, но кожух прочно сидит.
Работает за счёт лёгкого натяжения и геометрии.

**Для VikaVoice берём такой же принцип:**
- Кожух = внешняя тканевая рубашка + внутренний пластиковый каркас
- Съём для чистки/замены цвета — сжать + сдвинуть
- Без инструментов
- Пользователь может купить у нас «второй кожух другого цвета» как аксессуар

### 3. Реальный опыт 3D-печати ⏱️

Сообщество печатает такие кожухи на бытовых принтерах:

| Параметр | Значение |
|----------|----------|
| Принтер | Anycubic Kossel Plus (аналог — Bambu, Prusa) |
| Заполнение | 25% |
| Форма ячейки | 6-8-гранник |
| Время печати | **~16 часов** на один кожух |
| Постобработка | Долгое снятие поддержек |

**Для нашего плана:** учитываем ~16 часов печати на один кожух → для тестирования
5 вариантов ткани нужно 5 напечатанных кожухов = **80 часов принтера + материалы**.

**Оптимизация:** сначала печатаем **один** кожух и тестируем на нём все ткани
поочерёдно (натягиваем-снимаем). Проверяем что не рвётся и не деформирует пластик.

## Обновление плана для VikaVoice

С учётом опыта сообщества **обновляю подход к ткани** в
[INDUSTRIAL-DESIGN.md](INDUSTRIAL-DESIGN.md):

### Приоритет №1 — микрофонная прозрачность
Раньше писала «акустическая прозрачность». Правильнее — **не задушить микрофоны**.
Наш кейс: 4 микрофона ReSpeaker слушают через ткань пользователя, находящегося
в 2-3 метрах. Ткань не должна:
- Гасить речь на частотах 300-3400 Гц (человеческий голос)
- Увеличивать SNR на входе микрофонов
- Резонировать на wake-word частотах

### Тест-план: как выбрать ткань

1. Купить **3-5 образцов** (~2000₽ всего):
   - LEORY с AliExpress (1-2 варианта)
   - Автоакустическая с местного авторынка (1-2 варианта)
   - Плотная ткань «как у Яндекса» — контрольный образец
2. Записать эталонную речь с 2м от ReSpeaker **без ткани** (baseline)
3. Записать ту же речь **с каждой тканью** поверх массива
4. Замерить:
   - SNR (сигнал/шум)
   - Затухание на частотах голоса
   - Работоспособность wake-word (частота срабатывания)
5. Победитель = максимум SNR + минимум затухания + wake-word работает как без ткани

### Стартовый выбор для первого прототипа
LEORY с AliExpress + автоакустическая (тёмно-серая) с местного авторынка.
Итого: ~1500₽, доставка 1-2 недели или сразу.

## Съёмный цветной кожух — как продуктовая фишка

Из опыта Яндекса: **разные цвета кожухов продавались как аксессуары**.
Люди готовы платить за смену цвета под интерьер.

**Идея для нас (записать в roadmap):**
- Базовый кожух в комплекте (тёмно-серый / графит)
- Отдельно доп. кожухи: белый, бежевый, синий, тёмно-зелёный
- Цена дополнительного кожуха — 800-1500 ₽
- Простая маржа + узнаваемый продуктовый штрих (как разные ремешки у Apple Watch)

## Итого — что забираем к себе

| Идея | Применение |
|------|-----------|
| Bayonet-съём (сжатие + сдвиг) | Механика крепления кожуха |
| Отдельно продавать цветные кожухи | Продуктовая маржа + персонализация |
| Ткань LEORY с AliExpress | Стартовый образец для теста |
| Автоакустическая с авторынка | Дешёвый локальный вариант |
| Тест на **микрофонную** прозрачность | Не акустическую как у колонок |
| **Избегать плотных тканей** | Иначе задушим микрофоны |
| ~16 часов печати на кожух | Планирование прототипирования |
