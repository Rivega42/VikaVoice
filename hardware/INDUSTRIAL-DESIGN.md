## Референсы и что берём

### 🏆 Яндекс.Станция 2 — апериодическая решётка

**Источник:** [github.com/yandex/aperiodic_grille](https://github.com/yandex/aperiodic_grille)
**Лицензия:** **CC BY-SA 4.0** (Attribution-ShareAlike) — **можно использовать в коммерции**
при сохранении атрибуции и такой же лицензии на производную модель.

**Формат файлов:**
- `Aperiodic_grille.STEP.zip` — параметрическая CAD-модель (для доработки под наши размеры)
- `Aperiodic_grille.STL.zip` — готовая для 3D-печати
- `Aperiodic_grille.dxf` — векторный чертёж (для лазерной резки)
- `Aperiodic Grille.png` — рендер

**Разбор техрешения** (по [статье Хабра 673192](https://habr.com/ru/companies/yandex/articles/673192/)):

| Зона решётки | Форма отверстий | Зачем |
|---|---|---|
| Край | Шестигранные соты, толстая пластина | Прочность к физвоздействиям |
| Средняя зона | Апериодические, плавно тоньше | Максимальная акустическая прозрачность |
| Напротив динамика | Кольцо + апериодические | Не касается пульсирующего кольца динамика |
| Отдельные точки | Полуотверстия (0.1мм внутрь) | Однородность визуально под тканью |
| В шумных зонах | Заглушенные отверстия | Блокировка воздушного шума от компонентов |

**Замеренная выгода:** апериодический паттерн даёт **меньшую нелинейность АЧХ** по всем
частотам vs обычные соты. Разница 2-3 дБ по диаграмме направленности. Инженерное
подтверждение внешней лабораторией.

**Что делаем:** берём STEP-модель, масштабируем под наши размеры корпуса, адаптируем
центральное отверстие под наш динамик. Атрибуция → `hardware/GRILLE-ATTRIBUTION.md`
+ THIRD_PARTY_NOTICES.md.

### 🎯 Яндекс.Станция 1 — компоновка и элементная база

**Источник:** [mychooz.com/reviews/jandeks-stancija-28840](https://mychooz.com/reviews/jandeks-stancija-28840)

**Технические характеристики (референс для VikaVoice):**

| Параметр | Значение | Наш комментарий |
|----------|----------|-----------------|
| Габариты | 141 × 141 × 231 мм | Наш ориентир — цилиндр Ø150 × H180 |
| Вес | 2.9 кг (с кожухом), 2.5 без | Мы легче — ~1.5-2 кг |
| **Микрофоны** | **7 встроенных** | У нас 4-6 (ReSpeaker XMOS XVF3800) |
| ВЧ динамики | 2 × 10 Вт, 20 мм | У нас 1 динамик проще (голос+уведомления) |
| НЧ динамик | 1 × 30 Вт, 85 мм | Не нужен (мы не музыка) |
| Пассивные излучатели | 2 × 95 мм | Не нужны |
| Материалы корпуса | Алюминий + аудиоткань | Мы: пластик + аудиоткань |
| Питание | От сети (внешний БП) | RPi4 + USB-C 5V/3A |
| Процессор | AllWinner Cortex-A53 4×1.4 ГГц | У нас Broadcom BCM2711 4×1.5 ГГц (RPi4) |

**Ключевые выводы:**
- Яндекс делал **колонку с ассистентом**, у нас — **ассистент с колонкой**.
  → Радикально проще акустика: 1 динамик, никаких пассивных излучателей, никакого сабвуфера.
- **7 микрофонов** — избыточно для нашего кейса (у нас DSP на XVF3800 берёт beamforming).
- Металлическая вставка сзади — **радиатор**. У нас RPi4 в подходящих условиях выделяет
  ~5W, нужен активный кулер или большой пассив.
- **Съёмный кожух** — интересное решение (разные цвета продают как аксессуары). Заложить.

## Проработка корпуса VikaVoice

### Форм-фактор

**Вариант А (текущий, из CONCEPT.md):** цилиндр Ø150 × H180 мм.
Похож на Meeting Owl / Jabra Speak / Яндекс.Станция.

**Компоновка внутри (сверху вниз):**
```
┌─────────────────────┐
│ Экран 3.5" IPS      │  ← опционально, для транскрипта
│ + LED-кольцо        │  ← индикация состояния (WS2812B, 16 LED)
├─────────────────────┤
│ Кнопка приватности  │  ← физическое отключение микрофонов
│ (+ мут. светодиод)  │
├─────────────────────┤
│ ReSpeaker XMOS      │  ← 4-6 микрофонов по кругу, DSP XVF3800
│ XVF3800 4-Mic       │
├─────────────────────┤
│ Динамик 40-50 мм    │  ← широкополосный, 5-10 Вт
│                     │
├─────────────────────┤
│ RPi 4 8GB           │  ← мозг + активный кулер
│ + радиатор          │
├─────────────────────┤
│ БП USB-C            │  ← 5V/3A, внутрь корпуса
│ + разъёмы (Ethernet)│
└─────────────────────┘
```

### Апериодическая решётка — куда

Апериодический паттерн применяем **на всей боковой поверхности цилиндра** (360°):
- Микрофоны кругового массива слышат сквозь решётку
- Динамик излучает через решётку
- Ткань натянута поверх — визуально однородно

Наверху — сплошная крышка с экраном и LED-кольцом.
Снизу — сплошное дно с резиновыми ножками.

## Ткань — открытый вопрос ⚠️

Ром, вот тут **надо думать и тестировать**. Яндекс потратил месяцы на подбор
натяжения, клея, температурного режима — не с первого раза удалось.

**Требования к ткани:**

| Свойство | Зачем |
|----------|-------|
| Акустическая прозрачность | Не гасит звук из динамика, не глушит микрофоны |
| Плотное плетение | Не проступает решётка сквозь ткань |
| Пыле- и влагостойкость | Домашние условия, кухня, дети |
| Не проминается | Внешний вид долговременно |
| Натяжимость без «поплывшего» плетения | При натяжении не идёт полосами |
| Огнестойкость | Для B2B-сегмента желательно |
| Цвет и текстура | Продуктовый вид, брендинг |

**Варианты для исследования:**

### Кандидаты — реальные ткани для акустических решёток

| Ткань | Тип | Плюсы | Минусы | Источник |
|-------|-----|-------|--------|----------|
| **Yandex-подобная бежевая** | Плотное трикотажное плетение | Проверена в серии | Точный состав не публикуется | Разобрать б/у Станцию 2 |
| **Kvadrat Divina** | Мебельная шерсть | Премиум, огнестойкая | Дорого (€100+/м²) | Kvadrat.dk |
| **Camira Blazer** | Шерсть 100% | Огнестойкая, для B2B | Дорого | Camira.com |
| **Sonos-fabric** | Синтетика с акустическими свойствами | Хорошая прозрачность | Проприетарная | Разобрать Sonos |
| **Обычная акустическая ткань** | Полиэстер | Дёшево, доступно | Обычная, не премиум | Ozon, авторынки (обшивка потолков) |
| **Ткань от гитарных усилителей** | «Grill cloth» | Классика (Fender, Marshall) | Ретро, не всем подойдёт | Muzstore, гитарные магазины |

### Что делать

1. **Купить образцы 5-7 тканей** по 30×30 см (~500-2000 ₽ за образец)
2. Натянуть на **тестовый прототип** (просто цилиндр из 3D-печати с решёткой)
3. Замерить:
   - Акустическая прозрачность (микрофоном на разных частотах)
   - Визуальную однородность под разным светом
   - Тактильные ощущения (продукт же трогают)
4. Выбрать 1-2 победителей для прототипа

**Изначальный выбор для первого прототипа:** обычная акустическая ткань из авторынка
(тёмно-серая, ~500-1000 ₽/м) — быстро, дёшево, «более-менее».

**Для продуктовой версии:** Camira Blazer или Kvadrat (B2B) + отдельно потребительский
вариант — плотная синтетика с брендированным паттерном.

## Материалы корпуса

| Часть | Материал | Обоснование |
|-------|----------|-------------|
| Решётка (боковая) | ABS/PLA пластик для прототипа, потом ABS для серии | Прозрачность звука, апериодический паттерн 3D-печать |
| Крышка / дно | ABS пластик | Прочность, лёгкость |
| Декоративный кожух | ABS с soft-touch покрытием | Продуктовый вид |
| Внутренний каркас | Алюминий 3мм / жёсткий пластик | Крепление платы, радиатор |
| Ножки | Силикон / резина | Не скользит, гасит вибрации |
| Тканевое покрытие | См. раздел выше | ⚠️ Открытый вопрос |

## Технологии производства (по объёму)

**Прототип (1-10 шт):** 3D-печать FDM (Prusa/Bambu), ABS.
Cost: ~500-1500 ₽ материала на устройство + 8-15 часов печати.

**Малая серия (10-100 шт):** SLA-печать (Formlabs) или заказ формованных деталей.
Cost: ~1500-3000 ₽/устройство корпус.

**Средняя серия (100-1000 шт):** Литьё пластика на заказ (Китай, Россия).
Cost: NRE 100-500к ₽ на пресс-формы + ~300-800 ₽/устройство.

**Большая серия (1000+):** Полноценные пресс-формы, литьё в РФ или ЮВА.
Cost: NRE ~2-5 млн ₽ + ~200-500 ₽/устройство.

## Открытые вопросы

- [ ] **Ткань** — надо тестировать 5-7 образцов (см. выше)
- [ ] Точные габариты — зависят от финального выбора динамика

- [ ] Съёмный цветной кожух — как у Яндекс — заложить или нет?
- [ ] Как крепить ткань — клей / термо / рама-натяжитель?
- [ ] Есть ли смысл разбирать б/у Яндекс.Станцию 2 для изучения ткани и её крепления?

## Действия ближайшего этапа

1. Скачать `Aperiodic_grille.STEP` из репо Яндекса → адаптировать под наши размеры в Fusion 360/FreeCAD
2. Напечатать первый тестовый прототип решётки (без электроники)
3. Заказать образцы 5 тканей (~3000 ₽)
4. Натянуть → замерить → выбрать
5. Собрать первый визуальный макет (без работающей электроники, только корпус)
