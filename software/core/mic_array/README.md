# Микрофонный массив (наш слой, BUILD)

В Meetily отсутствует. Здесь живёт интерфейс к аппаратному звуковому тракту:
- многоканальный захват с кругового MEMS-массива;
- формирование направленного луча (beamforming);
- определение направления на источник (DOA) — помечает сектор говорящего;
- эхо/шумоподавление.

Прототип: готовый USB-массив с аппаратным DSP (XVF-3000) отдаёт уже обработанный
поток + метаданные направления. Серия: своя плата 6–8 MEMS на XVF-3800 (нужен NRE).

Референс по многоканальной обработке (VAD/ресемплинг) — core/vendor/meetily/.../audio
после запуска scripts/vendor_meetily.sh (код на Rust, под десктоп; переносим подход).
