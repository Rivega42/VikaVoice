# VikaVoice 🎤

> Умное голосовое устройство на базе RPi4: **домашний AI-ассистент** + **AI-фасилитатор встреч**.

Одна аппаратная платформа, два режима прошивки.

---

## Концепция

### Режим 1 — Домашняя Вика

Устройство стоит дома. Слушает wake-word «Вика». Отвечает голосом через локальный TTS.
Мозг — OCPlatform-инстанс (уже работает на сервере, память и личность сохраняются между разговорами).

**Что умеет:**
- Wake-word «Вика» → живой разговор голосом
- Помнит контекст, задачи, расписание
- Напоминает про воду, перерыв, встречи с детьми
- Читает вслух задачи и новости утром
- Работает без телефона — просто голосом в комнате

### Режим 2 — AI-Фасилитатор

Устройство ставится в центр стола на встрече. Слушает всех (360°).
Транскрибирует, понимает кто говорит, **активно участвует** — модерирует брейншторм, следит за таймингом, распределяет задачи.

**Что умеет:**
- Слушать круговой микрофон, различать участников (diarization)
- Транскрибировать в реальном времени
- Задавать уточняющие вопросы, подводить итоги
- В конце: протокол + список задач + отправка в Telegram/Notion
- Показывать транскрипт на экране рядом

### Режим 3 — Транскрайбер *(расширение, в разработке)*

Минималистичный вариант: только запись + транскрипция + протокол. Без активного участия AI.
Та же аппаратная платформа, другая прошивка.

---

## Почему это интересно как продукт

| Конкурент | Ограничение |
|-----------|-------------|
| Otter.ai | Только транскрипт, не участвует, англ. язык |
| Fireflies | Только для Zoom/Meet, не офлайн |
| tl;dv, Fathom | Видеозвонки, не физическое устройство |
| Яндекс.Транскрибация | Пассивно, без AI-участия |

**Активного AI-участника офлайн-встречи — нет.** Ниша пустая.

**Преимущества нашего подхода:**
- Локально (приватность — важно для юристов, врачей, гос. компаний)
- Русский язык из коробки (Whisper large-v3)
- Работает без интернета (STT + TTS локально)
- RPi4 + микрофон ≈ 20-25к ₽ vs подписки $30+/мес
