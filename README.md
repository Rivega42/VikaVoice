# VikaVoice 🎤

> Умное голосовое устройство на базе RPi4.
> **Одна Вика — несколько навыков.** Домашний ассистент, фасилитатор встреч, транскрайбер — всё это разные роли одной личности, а не разные прошивки.

---

## Концепция

Вика живёт в OCPlatform (память, характер, инструменты). VikaVoice — её **тело**: микрофон, колонка, RPi4 в комнате. Одна и та же Вика умеет вести себя по-разному в зависимости от контекста — это её навыки.

### Навыки

#### 🏠 Навык 1 — Домашний ассистент

Устройство стоит дома. Слушает wake-word «Вика». Отвечает голосом.

- Живой разговор без телефона, просто голосом в комнате
- Помнит контекст, задачи, расписание
- Напоминает про воду, перерыв, встречи с детьми
- Читает вслух задачи и новости утром

#### 🎯 Навык 2 — Фасилитатор встреч

Устройство ставится на стол. Круговой микрофон 360°, различает участников.

- Транскрибирует в реальном времени
- Активно участвует: модерирует брейншторм, задаёт вопросы
- Следит за таймингом, подводит итоги
- В конце: протокол + список задач + отправка в Telegram/Notion

#### 📝 Навык 3 — Транскрайбер *(в разработке)*

Минималистичный режим: только запись + транскрипция + протокол. Без активного участия.

- VAD + diarization ("Спикер 1", "Спикер 2")
- В конце — суммаризация и протокол
- Для тех, кому не нужен активный AI, только запись

#### 🔒 Навык 4 — Приватный режим

Голосовая команда **«Вика, приватный режим»** — и всё уходит в локальную обработку.

- STT — только whisper.cpp на RPi4 (никаких API)
- LLM — локальная модель на RPi4 (маленькая, но своя) или отключение LLM
- TTS — только Piper локально
- **Ничего не уходит в OCPlatform gateway и вообще в сеть**
- Индикация: красный LED на кольце
- Выход: «Вика, обычный режим» или таймер

**Зачем:** для юристов, врачей, психологов — когда клиент в комнате и данные не должны покидать устройство. Или просто когда обсуждаешь что-то, что не хочешь чтобы попало в облако.

## Почему это интересно как продукт

| Конкурент | Ограничение |
|-----------|-------------|
| Otter.ai | Только транскрипт, не участвует, англ. язык |
| Fireflies | Только для Zoom/Meet, не офлайн |
| tl;dv, Fathom | Видеозвонки, не физическое устройство |
| Яндекс.Транскрибация | Пассивно, без AI-участия |

**Активного AI-участника офлайн-встречи с локальным приватным режимом — нет.** Ниша пустая.

**Наши преимущества:**
- Локально (STT + TTS на устройстве, для приватного режима — вообще всё)
- Русский язык из коробки (Whisper large-v3)
- Живая личность (Вика, а не безликий ассистент)
- RPi4 + микрофон ≈ 24к ₽ vs подписки $30+/мес
